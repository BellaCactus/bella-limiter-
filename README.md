# Bella Limiter

A lightweight Windows network limiter / overlay tool used primarily for Destiny 2.

This project focuses on:
- low overhead
- fast toggling
- overlay-based control
- practical use during PvE / PvP activities

## Features
- Module-based limiting (PvE / PvP / API Block, etc.)
- Toggleable overlay
- Hotkey support
- Automatic resync options
- Designed for real-time use while gaming

## Status
This project is currently **personal-use / experimental**.
Documentation and cleanup will be added over time.

## Notes
- This tool was built to solve a specific problem and is shared as-is.
- Some functionality may be opinionated or game-specific.
